Kezdőlap

Ez a projekt egy többoldalas web alkalmazás, ami a WEB-programozás I. tantárgy beadandója is egyben. A weboldal külömböző frontend technológiák alkalmazásával valósítja meg az előadáson tanultakat. Az oldal tartalmaz CRUD táblázatkezelést, AJAX API-hívásokat, HTML5 API példákat, valamint React alapú alkalmazásokat.

A projekt fejlesztése csapatmunkában történt. A fejlesztési folyamat során GitHub verziókövetőt használtunk a forráskód kezelésére.

Az elkészült weboldal egy internetes tárhelyen lett publikálva, így bárki számára elérhető.
